🌐 Sistemas Operacionais

Um sistema operacional é o software que gerencia hardware e permite o uso do computador.

📱 iOS

Desenvolvido pela Apple, baseado no Darwin.

🤖 Android

Baseado no kernel Linux e desenvolvido pelo Google.

🐧 Linux

Sistema open-source criado por Linus Torvalds.

🪟 Windows

Sistema da Microsoft amplamente usado em PCs.
